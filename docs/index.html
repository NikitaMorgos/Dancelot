Dancelot

База знаний по мастер-классам для танцоров

Как пользоваться

  1. Открой бота @Dancelot_bot в Telegram.
  2. Отправляй рекапы: видео + подпись (стиль, уровень, навык, название, заметки).
  3. Команды: /list — список, /login — ссылка для входа на сайт.
Открыть бота в Telegram

Полный веб-интерфейс (вход + дашборд) работает при запуске проекта локально
или при развёрнутом бэкенде. Репозиторий на GitHub.

⚠️ На этой странице входа нет — это только информация. Чтобы по ссылке из /login можно было войти в базу рекапов, нужно развернуть бэкенд (см. раздел «Деплой» в README).
